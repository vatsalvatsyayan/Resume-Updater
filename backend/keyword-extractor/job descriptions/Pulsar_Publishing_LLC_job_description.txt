About the job

Junior Software Developer (Full Stack)

Experience: 1+ year

Location: Remote / Global Team

Type: Full-Time




About the Role

We are looking for a motivated Junior Software Developer to join our fast-paced, globally distributed engineering team. In this role, you will help maintain existing products, build new applications from the ground up, and collaborate closely with teammates through pair programming when needed. You’ll work autonomously on your tasks while contributing to high-impact software used by millions of users worldwide.




Responsibilities

Develop and maintain full-stack applications using Next.js, React, and TypeScript
Contribute to both front-end and back-end features
Work with SQL databases to build, optimize, and maintain data-driven applications
Participate in code reviews and follow best practices for version control using Git
Collaborate with a global team and communicate effectively in a fast-paced environment
Pair program with senior developers when necessary to solve complex problems
Maintain existing codebases and help build new applications from scratch
Take ownership of tasks and deliver high-quality, scalable solutions




Required Skills

1+ year of professional software development experience
Proficiency with:
Next.js
React
TypeScript
Git
SQL
Understanding of full-stack development principles
Ability to work independently and in a team setting




Nice-to-Have Skills

PHP experience
Python experience
UX Design knowledge or interest




Who You Are

A self-starter who enjoys learning and building
Passionate about creating high-impact, user-focused software
Comfortable working across time zones with a diverse global team
Resourceful, detail-oriented, and eager to grow your technical skillset
