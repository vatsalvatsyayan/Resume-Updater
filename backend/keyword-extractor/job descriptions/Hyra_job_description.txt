About the job

Entry Level Software Developer | Salary: £75,000 - £100,000 annual




Hyra is working on behalf of Property Matrix to hire for the position of Entry Level Software Developer. This role involves contributing to real product development by building and improving features for a cloud-based property management platform. You will work within a collaborative team to ship high-quality software while gaining exposure across the full technical stack.




What you'll be doing

• Develop and improve product features for residential and commercial real estate operators

• Contribute to full-stack development tasks alongside a collaborative engineering team

• Perform software testing and debugging to ensure high-quality code deployment

• Build practical solutions to improve user experience and business outcomes

• Iterate on existing workflows including tenant portals and maintenance systems




What we're looking for

• Foundational knowledge of software engineering principles and problem-solving

• Ability to write, test, and debug code in a professional development environment

• Eagerness to learn and take ownership of technical tasks and features

• Strong communication skills for working within a collaborative team setting

• Must be legally authorized to work in the United States




What Property Matrix offers

• Professional growth path within a supportive development environment

• Exposure to full-stack development and modern cloud-based platforms

• Opportunity to work on enterprise-level software used by real customers

• Collaborative team culture focused on long-term success
