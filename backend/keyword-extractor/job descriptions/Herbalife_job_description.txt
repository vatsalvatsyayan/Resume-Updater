About the job

Overview

THE ROLE:

The Application Developer I is an entry-level engineering role focused on building and maintaining applications, APIs, and web interfaces under guidance. This position supports the development lifecycle by implementing features, fixing bugs, and learning guidelines in software engineering. Ideal candidates are eager to grow technical skills and collaborate within a team environment.

HOW YOU WOULD CONTRIBUTE: 


 Compose, build, and maintain full-stack applications, bringing to bear backend technologies including Python, Node.js, and .NET to deliver scalable and efficient solutions.
 Implement front-end features and user interfaces using HTML, CSS, JavaScript, and frameworks like React, Shadcn, Typescript to ensure responsive and accessible design.
 Write clean, maintainable code following team standards and participate in code reviews.
 Collaborate with senior developers to fix and resolve issues.
 Maintain documentation for assigned components and features.
 Know the latest programming concepts and new technologies.
 Perform other related duties as assigned.



Qualifications

SKILLS AND BACKGROUND REQUIRED TO BE SUCCESSFUL: 


 0–3 years of professional or academic experience in software development.
 Basic knowledge of one or more programming languages (Python, Node.js, Java, or .NET).
 Understanding of web technologies: HTML, CSS, JavaScript.
 Familiarity with RESTful APIs and database concepts.
 Strong willingness to learn and adapt to new technologies.
 Effective communication and teamwork skills.



Experience:


 0–3 years in software development or related internships.
 Experience with version control systems (e.g., Git) and collaborative development workflows (branching, pull requests, code reviews).



Education:


 Bachelor’s Degree in Computer Science, Engineering, or related field (or equivalent practical experience).
